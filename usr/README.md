# YAI userspace surface

This subtree hosts the install-facing userspace layer:

- `usr/bin` entrypoints
- `usr/include/yai` public userspace headers
- `usr/lib/yai` implementation libraries
- `usr/share` manpages, completions, prompts, themes, docs, examples

Canonical client-facing API lives under `usr/include/yai/client`.
Shell-facing public headers live under `usr/include/yai/shell`.
