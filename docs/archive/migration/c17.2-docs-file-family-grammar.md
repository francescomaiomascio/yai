# C17.2 Docs File-Family Grammar

## Live Docs Grammar Contract

This grammar applies to live docs (`docs/**` except `docs/archive/**`).

## Allowed Terminal Classes

### Architecture

- `*-architecture.md`
- `*-topology.md`
- `*-flow.md`
- `*-state.md`
- `*-boundary.md`
- `*-integration.md`

### Guides

- `*-guide.md`
- `*-workflow.md`
- `*-walkthrough.md`
- `*-checklist.md`
- `*-debugging.md`

### Runbooks

- `*-runbook.md`
- `*-procedure.md`
- `*-qualification.md`
- `*-expected-outputs.md`

### Reference

- `*-reference.md`
- `*-contracts.md`
- `*-schemas.md`
- `*-registries.md`
- `*-commands.md`
- `*-sdk.md`

### Program

- RFC files: `rfc-<domain>-<nnn>-<slug>.md`
- ADR files: `adr-<domain>-<nnn>-<slug>.md`
- MP files: `mp-<domain>-<nnn>-<slug>.md` or `mp-<domain>-<nnn>-<slug>-vX-Y-Z.md`
- Other program docs: `*-policy.md`, `*-report.md`, `*-matrix.md`, `*-template.md`

## Disallowed Live Suffixes

- `*-model.md`
- `*-baseline.md`
- `*-closeout.md`
- `*-refoundation.md`
- `*-legacy-notes.md`
- `*-residue.md`

## Archive Exceptions

Archive files may retain historical naming when needed for migration traceability.
