# Findings

Use this folder for atomic audit findings, one file per finding.

Naming suggestion:
- `FINDING-001-<short-title>.md`

Minimum fields per finding:
- `id`
- `severity`
- `affected_repo`
- `evidence`
- `recommended_action`
- `verification`
