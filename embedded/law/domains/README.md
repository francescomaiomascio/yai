# Domains Layer

`law/domains` is the canonical home of executable domain normativity.

Each domain module includes:
- discovery signals and heuristics
- action/asset/risk model
- policy packs (baseline, deny, quarantine, review)
- provider/command/resource mappings
- evidence requirements and trace templates

`ops` continues to host catalog, rollout, qualification, and evidence linkage material.
`ops` packs are descriptive/qualification-facing and are no longer authoritative for normative execution logic.
